FloodRisk2¶

Cos’è FloodRisk2 ?

FloodRisk2 è un plugin per QGIS 3.x ed è una nuova versione del precedente FloodRisk per QGIS 2.x, con funzionalità aggiuntive.

Esso consente la valutazione delle conseguente, in termini di perdite di vite e di danni economici diretti ed inoltre consente la valutazione quantitativa dei benefici in termini di riduzione del danno come risultato di misure di mitigazione.

Il termine RISCHIO : Rischio = (Probabilità) x (Conseguenze)

Il Plug-in¶

Contenuti:

  • Project Management
  • Database Manager
    • Esempio di tabella di Tipi di Bene
    • Esempio di tabella di ListaTipiCurveAltezzaDanno
    • Esempio di tabella di curva Altezza-Danno
    • Esempio di tabella di FatalityRates (Tasso di Perdite di Vite)
    • Esempio di tabella FloodSeverity (Intensità dell’inondazione)
  • Danni Economici Diretti
  • Scenario perdite di vite (LOL)
  • Scenari Analisi Costi Benefici
    • Il tab Valutazione scenari
    • Il tab Analisi Costi Benefici
